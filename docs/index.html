Which one should I read?

I need to run something

Start with the operator guide. Stop reading when the guide tells you to jump to the deeper docs.

I am debugging or extending

Use the system reference. It is the "what exists and why it matters" document.

I want to build my own replica

Use the museum design notes. That is the builder-facing reference, not an operator checklist.
